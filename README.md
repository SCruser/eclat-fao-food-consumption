# Association Rule Mining on FAO Food Consumption Data

## Project Overview
This project analyzes food consumption patterns using association rule mining techniques.
The ECLAT algorithm is used as an alternative to Apriori due to its efficiency on sparse datasets.

## Data Source
FAO GIFT - Individual Food Consumption Data:
https://www.fao.org/gift-individual-food-consumption/data/en

## Algorithms
- ECLAT
- Apriori (for comparison)

## Tools and Libraries
- Python
- pandas
- numpy
- mlxtend
- matplotlib
